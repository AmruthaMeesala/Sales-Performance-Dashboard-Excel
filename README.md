# Sales Performance Dashboard (Excel)

## Project Overview

This project presents an interactive sales dashboard built entirely in Microsoft Excel.

The dashboard provides insights into sales performance, product performance, regional sales distribution, and business trends using Pivot Tables, Pivot Charts, Slicers, and Excel formulas.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas

## Key KPIs

* Total Sales
* Total Orders
* Average Sales
* Top Performing Products
* Regional Sales Distribution

## Dashboard Features

### Sales Overview

Displays overall sales performance through KPI cards.

### Product Analysis

Identifies top-selling and low-performing products.

### Regional Analysis

Compares sales performance across different regions.

### Trend Analysis

Tracks sales performance over time.

### Interactive Filtering

Uses slicers to dynamically filter dashboard visuals.

## Key Insights

* Certain products contribute significantly to overall revenue.
* Regional sales performance varies considerably.
* Sales trends reveal seasonal fluctuations.
* A small number of products generate a large share of total sales.
